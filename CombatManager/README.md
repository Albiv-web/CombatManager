# CombatManager

CombatManager adds a standalone AI movement sandbox for From The Depths.

Open it with `Ctrl+Shift+C`. The overlay shows an opaque Red-centered 2D
tactical grid where Blue and Red both have mainframe-like behaviour and
manoeuvre settings. Both sides plan against each other from the same tick
snapshot, then move using shared ship/tank, hover, six-axis, or airplane
movement-card approximations. `Import Blue AI` can copy supported settings from
a selected mainframe on the focused craft once; Red remains manually configured.

Version: 0.1.5
