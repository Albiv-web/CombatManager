# CombatManager

CombatManager adds a standalone AI movement sandbox for From The Depths.

Open it with `Ctrl+Shift+C`. The overlay shows an opaque target-centered 2D
tactical grid with circle, point-at, and broadside presets. `Import Current AI`
can copy supported settings from the focused craft once, but the sandbox then
runs independently and remains read-only toward the real craft.

Version: 0.1.2
