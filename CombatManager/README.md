# CombatManager

CombatManager adds a standalone AI movement sandbox for From The Depths.

Open it with `Ctrl+Shift+C`. The overlay shows an opaque target-centered 2D
tactical grid with scenario presets, target profiles, moving-target paths, raw
AI steer bearings, finite motion points, and movement-card-specific craft
pursuit for circle, point-at, broadside, and Naval 2.0 presets. `Import Current
AI` can copy supported settings from a selected mainframe on the focused craft
once, but the sandbox then runs independently and remains read-only toward the
real craft.

Version: 0.1.4
